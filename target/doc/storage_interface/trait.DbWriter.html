fn finalize_state_snapshot(
    &self,
    version: Version,
    output_with_proof: TransactionOutputListWithProof,
    ledger_infos: &[LedgerInfoWithSignatures]
) -> Result<()>

Finalizes a state snapshot that has already been restored to the database through a state snapshot receiver. This is required to bootstrap the transaction accumulator, populate transaction information, save the epoch ending ledger infos and delete genesis.

Note: this assumes that the output with proof has already been verified and that the state snapshot was restored at the same version.
